When building campaigns for StowStack clients:

1. Always structure campaigns by vacant unit type — never one-size-fits-all
2. Map audiences to demand triggers: moving, divorce, downsizing, remodel, estate cleanout, business overflow, college, vehicle storage, seasonal
3. Always include both prospecting and retargeting ad sets
4. Creative angles must be specific to the facility and market — no generic storage copy
5. Lead forms should qualify for: intent, location, unit type, timeline
6. Budget allocation default: 70% prospecting, 30% retargeting (adjust based on pixel maturity)
7. Naming convention: SS_{FacilityCode}_{UnitType}_{Audience}_{Objective}
8. Always recommend a tracking setup: Meta Pixel + Conversions API
9. Landing pages should be facility-specific, not the facility's existing website (unless it's already strong)
10. Every creative brief needs a clear CTA tied to a specific offer or unit type
11. Headline max: 40 characters. Primary text max: 125 characters. Be tight.
12. Prioritize unit types by vacancy severity — attack the worst performers first
